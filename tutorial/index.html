A tutorial Template to demonstrate Play Application with Play 2.5(Scala) and Pure.css

Introduction

A tutorial template if you want to develop application frontend with Pure.css.

This template will help you building a variety of Web application's frontend design using Pure.css which is a set of small, responsive CSS modules that you can use in every Play Application. Pure is extremely small css. The entire set of modules fits in at 3.8KB* minified and gzipped. It is designed with mobile devices in consideration. Pure is highly responsive, so elements look great on all screen sizes.

Pure Components

    In this template, the following common components designing using pure.css have been discussed. You can simply re-use these components in your application to build complex designs.
  • Forms
  • Table
  • Buttons
  • Grids
